## 安装Python

### 第1步：下载Python安装程序

1. 打开浏览器，访问Python官网：https://www.python.org/downloads/
2. 点击黄色的"Download Python 3.x.x"按钮（x.x表示最新版本号）
3. 安装文件会自动下载到您的电脑上

### 第2步：安装Python

1. 找到并双击下载的安装文件（通常在"下载"文件夹中）
2. 在安装界面上，**务必勾选**"Add Python to PATH"选项
3. 点击"Install Now"（现在安装）按钮
4. 等待安装完成
5. 看到"Setup was successful"（安装成功）的提示后，点击"Close"（关闭）按钮

### 第3步：验证安装

1. 按下`Win + R`键，打开"运行"对话框
2. 输入`cmd`并点击"确定"，打开命令提示符窗口
3. 在命令提示符窗口中输入：`python --version`并按回车键
4. 如果看到类似`Python 3.x.x`的输出，说明安装成功

## 安装必要的库

Excel去重工具需要安装pandas库才能正常工作。请按照以下步骤安装：

1. 按下`Win + R`键，打开"运行"对话框
2. 输入`cmd`并点击"确定"，打开命令提示符窗口
3. 在命令提示符窗口中输入以下命令并按回车键：
   ```
   pip install pandas openpyxl
   ```
4. 等待安装完成，会看到"Successfully installed"（安装成功）的提示

## 使用去重工具

### 交互式模式

交互式模式是最简单的使用方式，程序会引导您完成所有步骤。

1. 将您要检查的Excel文件放在与`check_duplicates.py`相同的文件夹中
2. 按下`Win + R`键，打开"运行"对话框
3. 输入`cmd`并点击"确定"，打开命令提示符窗口
4. 使用`cd`命令导航到`check_duplicates.py`所在的文件夹，例如：
   ```
   cd C:\Users\您的用户名\Desktop
   ```
5. 输入以下命令并按回车键启动交互式模式：
   ```
   python check_duplicates.py -i
   ```
6. 按照程序提示进行操作：
   - 选择要检查的Excel文件
   - 选择要检查的列（A、B、C等）
   - 查看检查结果
   - 选择是否保存结果到文件
   - 选择是否检查其他列

### 命令行模式

如果您已经知道要检查哪个文件和哪一列，可以直接使用命令行模式：

1. 按下`Win + R`键，打开"运行"对话框
2. 输入`cmd`并点击"确定"，打开命令提示符窗口
3. 使用`cd`命令导航到`check_duplicates.py`所在的文件夹
4. 输入以下命令并按回车键：
   ```
   python check_duplicates.py 您的Excel文件名.xlsx -c A
   ```
   其中，`您的Excel文件名.xlsx`是您要检查的Excel文件名，`A`是要检查的列（可以是A、B、C等列字母）

## 常见问题解答

### 问题1：运行程序时提示"python不是内部或外部命令"

**解决方法**：重新安装Python，确保在安装时勾选"Add Python to PATH"选项。

### 问题2：程序提示"ModuleNotFoundError: No module named 'pandas'"

**解决方法**：按照上面的"安装必要的库"部分的步骤安装pandas库。

### 问题3：如何检查Excel文件中的多个列？

**解决方法**：在交互式模式下，程序会在检查完一列后询问您是否要检查其他列。选择"y"并输入新的列字母即可。

### 问题4：如何理解检查结果？

**解决方法**：检查结果会显示每个重复值出现的行号。例如，"值 'ABC' 在A列中重复出现，行号为: [2, 5, 8]"表示值"ABC"在第2行、第5行和第8行重复出现。

### 问题5：如何保存检查结果？

**解决方法**：在交互式模式下，程序会询问您是否要将结果保存到文件。选择"y"并输入文件名（或使用默认文件名）即可。
